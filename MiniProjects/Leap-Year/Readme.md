# Leap Year Program
## Simple Leap Year Program To Check whether the entered year is Leap Year ✔ Or Not ❌.

- Enter The Year 🔽
 ```
Which year do you want to check? 2004
 ```

- Will Check Whether it is Leap Year ✔ Or Not ❌.
 ```
Leap Year. 
 ```
- Enter The Year Year 🔽
```
Which year do you want to check? 2005
```

- Result ⏳
```
Not Leap Year.
```
